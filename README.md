# tabnews-clone

Implementação clone do htpps://www.tabenews.com.br

## Como rodar

Recomendo rodar no codespaces

### Settar a versão do node:

```sh
nvm install
```

### Subir os serviços

```sh
npm run services:up
```

### Rodar localmente

```sh
npm run dev
```

### Rodar testes

Primeiro rodar em um terminal:

```sh
npm run dev
```

Em outro terminal rodar:

```sh
npm run test
```
